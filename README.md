## 📁 Estructura del repositorio
- `aed1/` – Algoritmos y Estructuras de Datos 1 (C++)
- `aed2/` – Algoritmos y Estructuras de Datos 2 (C++)
- `alf/` – Autómatas y Lenguajes Formales (Python)
